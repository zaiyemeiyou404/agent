# 西小豆课程查询 — 使用说明

## 文件说明

| 文件 | 作用 |
|------|------|
| `西小豆 (1).html` | **前端界面** — AI Agent 智能助手的完整Web应用 |
| `classtable_server.py` | **后端服务** — 课程查询的Python服务器 |

## 使用方法

### 第一步：启动后端服务

```bash
# 进入文件夹
cd G:/projects/测试

# 安装依赖（首次运行需要）
pip install flask flask-cors requests pillow numpy pycryptodome

# 启动服务
python classtable_server.py
```

服务会在 `http://localhost:5000` 运行，看到以下输出说明启动成功：

```
============================================================
  西小豆课表查询后端服务
  基于 XDYou 项目原理实现
============================================================

  服务地址: http://localhost:5000
  API 端点:
    POST /api/classtable - 获取课表
    GET  /api/health     - 健康检查
```

### 第二步：打开前端页面

直接用浏览器打开 `西小豆 (1).html`，然后：

1. 点击左侧 **"校园服务"** 模块展开
2. 点击 **"课表查询"**
3. 输入西电统一身份认证的 **学号** 和 **密码**
4. 选择身份类型（本科生/研究生）
5. 点击 **"登录并查询课表"**

## 工作原理

```
用户在网页输入学号密码
        ↓
前端发请求到 localhost:5000/api/classtable
        ↓
Python后端模拟登录 ids.xidian.edu.cn
        ↓
自动破解滑块验证码（NCC模板匹配）
        ↓
AES加密密码，提交登录表单
        ↓
获取 ehall.xidian.edu.cn 课表数据
        ↓
返回JSON给前端渲染课表
```

## 技术要点

### 1. IDS 统一身份认证登录
- 解析登录页隐藏字段
- AES-CBC 加密密码（IV: `xidianscriptsxdu`）
- 自动识别并破解滑块验证码（NCC模板匹配算法）

### 2. Ehall 教务系统对接
- 激活课表应用（appId: 4770397878132218）
- 获取当前学期代码与开学日期
- 调用 xskcb.do 接口获取课表数据

### 3. 数据解析
- 解析课程名称、教师、教室、节次、周次
- 周次使用位图存储（"1" 表示该周有课）
- 结构化输出供前端渲染

## 常见问题

### Q: 提示"连接服务器失败"
A: 检查后端服务是否启动，终端是否显示服务运行信息

### Q: 提示"验证码破解失败"
A: 可能是网络不稳定，稍后重试

### Q: 提示"账号或密码错误"
A: 检查学号密码是否正确，可以先在 ids.xidian.edu.cn 手动登录测试

### Q: 课表显示为空
A: 可能是当前学期课表尚未发布，或选课未完成
